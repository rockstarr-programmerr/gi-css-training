2. Cách nhúng CSS

Dương Trung

- 31/10/2019 -

Có 3 cách nhúng CSS:

  • External (phổ biến nhất)
  • Internal
  • Inline (hạn chế dùng)

Ví dụ:

  • External:
    • <link rel="stylesheet" href="../common.css">
  • Internal:
    • <style> .fun-class { display: block; } </style>
  • Inline:
    • <p style="text-transform: uppercase;">Dương Trung</p>
